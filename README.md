# Müzik Öneri Sistemi

Bu proje, kullanıcının duygularını analiz ederek kişiselleştirilmiş müzik önerileri sunan bir web uygulamasıdır. Makine öğrenimi modelleri kullanarak duyguları sınıflandırır ve uygun müzikleri önerir.

## Özellikler
- Duygu analizi ve sınıflandırma
- Kişiselleştirilmiş müzik önerileri
- Flask backend ve React frontend

## Kurulum ve Çalıştırma
1. Backend için: `cd backend && python flask_app.py`
2. Frontend için: `cd frontend && npm install && npm run dev`

## Teknolojiler
- Backend: Flask, Python, Scikit-learn
- Frontend: React, TypeScript, Vite
